Eventos de Teclado

  • Quando uma tecla do teclado é pressionada, o navegador dispara um evento "keydown". Quando a tecla é liberada, você recebe um evento "keyup".
  • O evento "keydown" é disparado não apenas quando a tecla é fisicamente pressionada, mas também toda vez que a tecla se repete enquanto é mantida pressionada. Isso pode causar problemas, como adicionar múltiplos elementos se uma tecla for pressionada por muito tempo.
  • O exemplo a seguir altera a cor de fundo da página para violeta quando a tecla "v" é pressionada e a reverte quando a tecla é liberada:

Esta página fica violeta quando você mantém a tecla V pressionada.

Código do Script:


<script>
  window.addEventListener("keydown", event => {
    if (event.key == "v") {
      document.body.style.background = "violet";
    }
  });
  window.addEventListener("keyup", event => {
    if (event.key == "v") {
      document.body.style.background = "";
    }
  });
</script>
  • O nome da tecla é acessado pela propriedade key do objeto de evento. Essa propriedade armazena uma string que, para a maioria das teclas, corresponde ao que a tecla digitaria. Teclas especiais como enter têm strings específicas como "Enter".
  • Teclas modificadoras como shift, ctrl, alt e meta (command no Mac) também geram eventos de tecla. Você pode verificar se essas teclas estão pressionadas usando as propriedades shiftKey, ctrlKey, altKey e metaKey.
  • Quando o usuário digita texto, usar eventos de tecla para determinar o que está sendo digitado pode ser problemático, especialmente com softwares de edição de método de entrada (IME) que combinam múltiplas teclas para criar caracteres.
  • Para detectar quando algo foi digitado, elementos que podem receber entrada, como <input> e <textarea>, disparam eventos "input" sempre que o conteúdo é alterado.
    • 
        <label for="textInput">Digite algo:</label>
        <input type="text" id="textInput">
        <p id="output"></p>
        
        <script>
          const input = document.getElementById('textInput');
          const output = document.getElementById('output');
        
          input.addEventListener('input', () => {
            output.textContent = `Você digitou: ${input.value}`;
          });
        </script>

Pressione Control-Espaço para continuar.

Código do Script:


<script>
  window.addEventListener("keydown", event => {
    if (event.key == " " && event.ctrlKey) {
      alert("Continuando!");
    }
  });
</script>
