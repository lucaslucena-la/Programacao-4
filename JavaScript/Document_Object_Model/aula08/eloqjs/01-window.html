Resumo: Manipuladores de Eventos

  • Interface básica exige leitura constante do estado das teclas para capturar eventos.
  • Máquinas primitivas usam essa abordagem, mas é ineficiente e arriscada.
  • Uma melhoria é o sistema operacional ou hardware colocar eventos em uma fila para serem processados.
  • Essa técnica, chamada de polling (consulta), pode tornar o software pouco responsivo.
  • Mecanismo preferido: o sistema notifica ativamente o código quando um evento ocorre.
  • Navegadores permitem registrar funções como manipuladores para eventos específicos.
