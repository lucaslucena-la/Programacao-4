Eventos de Foco

  • Quando um elemento recebe foco, o navegador dispara um evento "focus". Quando o foco é perdido, o evento "blur" é disparado.
  • Esses eventos não se propagam. Um manipulador no elemento pai não é notificado quando um elemento filho ganha ou perde foco.
  • O exemplo a seguir exibe um texto de ajuda para o campo de texto que está com foco:

Código do Script:


<script>
  let help = document.querySelector("#help");
  let fields = document.querySelectorAll("input");
  for (let field of Array.from(fields)) {
    field.addEventListener("focus", event => {
      let text = event.target.getAttribute("data-help");
      help.textContent = text;
    });
    field.addEventListener("blur", event => {
      help.textContent = "";
    });
  }
</script>
  • O objeto window também recebe eventos "focus" e "blur" quando o usuário alterna entre abas ou janelas do navegador.
