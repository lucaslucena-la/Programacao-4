Objetos de Evento

  • Embora não tenhamos abordado isso até agora, funções manipuladoras de eventos recebem um argumento: o objeto de evento.
  • Este objeto contém informações adicionais sobre o evento.
  • Por exemplo, para saber qual botão do mouse foi pressionado, podemos verificar a propriedade button do objeto de evento.

Código do Script:


<script>
  let button = document.querySelector("button");
  button.addEventListener("mousedown", event => {
    if (event.button == 0) {
      alert("Botão esquerdo");
    } else if (event.button == 1) {
      alert("Botão do meio");
    } else if (event.button == 2) {
      alert("Botão direito");
    }
  });
</script>
  • A informação armazenada em um objeto de evento varia conforme o tipo de evento.
  • A propriedade type do objeto sempre contém uma string que identifica o evento (como "click" ou "mousedown").
